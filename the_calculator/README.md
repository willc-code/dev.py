# Welcome to the Calculator!

About:
A python CLI calculator with the addition, subtraction, multiplication. division and continuous calculation with more than two numbers. 

## How to use 

Run `python main.py` and follow the prompts to input the relevant responses. 

## What it does

The program gets two numbers and selection of operation (addition, subtraction, multiplication. division) from the user. 
Once the calculation is done, the user can choose to continue the calculation with value from the previous answer or otherwise.
