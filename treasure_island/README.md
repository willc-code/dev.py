# Treasure Island Game

About: An interactive CLI treasure hunting game on Treasure Island. 

## How to use 

Run `python main.py` and follow the prompts to input the relevant responses. 

## What it does

It takes the user on a treasure hunting journey on the Treasure Island. Users will be given options to choose their steps... but only one path leads to the treasure......

### Learning outcomes 

- if/elif/else statements
- nested if statements 
- Use the backslash \ as an escape character to prevent an apostrophe from prematurely terminating a string.
e.g., print('she\'s saying "hello"')
- using .lower() and .upper() to convert all inputs into either upper or lower case. This is to ensure that same responses that differ in upper or lower case will be interpreted the same way. e.g. "Right" and "right" will both trigger the same response.
