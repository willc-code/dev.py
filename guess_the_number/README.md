# Guess The Number!

About: A simple python CLI game of guessing the number. 

## How to use 

Run `python main.py` and follow the prompts to input the relevant responses. 

## What it does

It randomly generates a number between 1 to 9. Make your guess accordingly. 
It tells your whether you got it right or not, and the number of attempts you took. 

### Learning outcomes

- practicing while loop - to keep the game going as long as the condition was not met, and breaking it when the guess is right.
