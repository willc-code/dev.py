# The Password Generator 

About: A password generator program built in Python

## How to use 

Run `python main.py` and input your answers following the prompts. 

## What it does

The password generator randomly generates password in letters, numbers and symbols.
Users will be able to input number of characters in each of the component (letters, numbers and symbols), and the password will be generated accordingly.

#### Learning Outcomes:
 - random.choices function, especially the "k" which is an integer defining the length of the returned list
 - random.shuffle function and the fact that this is only usable for lists and not string.
 - .join function to join the list into a string
 - "" in the .join function determines what is between the join. "" means nothing in between, "-" means - is in between e.g. 1-2-3
